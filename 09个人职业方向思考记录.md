### 四、我的过去与心结

#### 1. 从全校前十到普通

小学初中一直名列前茅，全校前十。高中之后变普通了。这个落差到现在都没消化——不是"接受了自己普通"，是"被迫接受了但心里不甘心"。

#### 2. 报志愿的遗憾

报志愿选了哈理工，觉得如果报南方说不定更好。认为这是"选错了，限制了发展"。每次做选择都会想起这件事，怕又选错。

#### 3. 培训班的沉没成本

大二报了科林明伦就业培训班，花了两万多，每周六日学6小时。但很多时候就是照着抄代码，没有真正理解。现在觉得"如果当时不报班就好了，我现在绝对义无反顾地考公务员"。培训班说坚持就一定能找到15万总包的工作，一直让我坚持。

#### 4. AI带来的无力感

有了AI之后虽然提效，但总感觉没有进步了，非常依赖。觉得AI比自己代码能力强很多，自己付出了很多但没有用。

### 六、三条路的分析

#### 路线A：私企技术岗

| 维度 | 评估 |
|------|------|
| 我能拿到的薪资 | 6-8k（上限1万，需补Java+项目+面试） |
| 培训班说的15万总包 | 有可能但概率约20-30%，不是"坚持就一定有" |
| 需要补的 | Java基础、Spring Boot概念、算法刷题、八股文，至少2-3个月 |
| 入职后 | 技术要持续学，考核有压力，稳定性差 |
| 长期 | 35岁问题是真实存在的，尤其对女性 |
| 适合我吗 | 不太适合。我的性格受不了高压和不确定性 |

**关于"工作到底有多累"：** 不是所有私企都996。月薪1万的私企大部分是正常上下班，偶尔加班。但累不累不只是加班时长，还有不确定性（裁员、倒闭、降薪）、绩效压力、随时待命、没有安全感。这些才是我性格受不了的部分。

**关于秋招：** 秋招确实是进私企最容易的时候，岗位比春招多70%。如果秋招不投私企，春招会更难。

#### 路线B：央国企科技岗

| 维度 | 评估 |
|------|------|
| 目标岗位 | 四大行省分行科技岗、三大运营商省公司、国网信通、政策性银行 |
| 需要补的 | 行测（与考公重合）、计算机基础（笔试考） |
| 入职后 | 技术更新慢，工作稳定，但工资天花板低 |
| 优势 | 8月秋招就开始，比国考早出结果3个月 |
| 笔试 | 考行测+计算机基础，与备考内容完全重合，不需要额外准备 |
| 面试 | 不考八股文，不考算法，能讲清楚项目就行 |
| 适合我吗 | 适合。稳定、压力小、不需要追新技术 |

**关键点：秋招投央国企=参加秋招=没有错过秋招。** 央国企秋招和私企秋招同时进行，投央国企不需要准备八股文，备考行测就是在准备央国企笔试。

#### 路线C：公务员（省考为主）

| 维度 | 评估 |
|------|------|
| 目标岗位 | 省会城市/二线城市，限计算机专业的岗位 |
| 需要补的 | 行测+申论，6-9个月系统备考 |
| 竞争 | 限专业+限户籍的基层岗实际竞争5-15人；省会城市竞争更大 |
| 入职后 | 最稳定，工资4-6k，在城市生活够用 |
| 适合我吗 | 适合。确定性高，焦虑少 |

**关于计算机岗分数线：** 计算机岗的进面分数线经常比三不限还高，因为报考的都是理工科，行测底子好，"人少但精"。但报计算机岗，我的行测水平在考生中是中位数以上；报三不限，跟文科生比言语没优势，跟法学生比常识没优势。所以继续报限计算机专业的岗位。

**关于"全民考公"：** 2025年国考报名303万人，但实际缴费220万，参加考试180万，认真备考3个月以上约60万，备考6个月以上约20万。303万里绝大多数是陪跑的。"全民考公"对认真备考的人来说是好消息——跟风的人越多，炮灰越多，我的相对排名越高。

**关于跨省考公：** 我不想回县城，想待在有盒马的城市。很多省份省考不限户籍（尤其对计算机专业），可以考大连、沈阳、济南、长沙、合肥等城市的岗位。跨省考=不限户籍=竞争更大=需要更高的分数。

### 七、我的纠结根源

纠结了三个月，表面问题是"私企还是体制内"，**真正的问题是：**

#### 1. 怕选错——因为觉得自己已经选错过一次了

报志愿没去南方，觉得选错了。每次做选择都怕"这次又选错"。

#### 2. 沉没成本——两万块和一年的周末

培训班花了两万多，每周六日6小时。如果不去找工作，觉得这些全白费了。但不管怎么选，那两万块都已经花了，回不来了。

#### 3. 不想"努力了但没用"

报志愿选错了觉得努力白费，AI比我会写代码觉得学编程白费，所以纠结三个月不敢选——怕这次又白费。

#### 4. "全或无"思维

觉得要么100%只备考，其他什么都不做；要么就不算全力，就考不上。实际上每天学6小时坚持9个月=1620小时，远超上岸所需的800-1000小时。

#### 5. 不确定自己到底想要什么

嘴上说选县城公务员，心里想去有盒马的城市生活。这种矛盾让选择变得更难。

### 八、关键认知转变

#### 1. 我需要的不是稳定，是确定感

我不是"受不了高压"，我在"证明自己"这件事上抗压能力很强（英语老师觉得我不行我就拼命学）。我需要的是"努力不会白费"的确定感。考公恰好是这种事——分数到了就能上，没有志愿填错、没有AI替代、没有市场变化。

#### 2. 考公是另一个"高中物理"

高中物理：明确目标（高考需要）+ 确定回报（学了多少考多少）= 死磕到底。

考公：明确目标（行测申论考高分）+ 确定回报（分数到了就上岸）= 同样可以死磕到底。

#### 3. 后悔是不可避免的

任何选择都会后悔——只能过一种人生，永远会好奇另一种人生。但"不选"才是最糟糕的选择，因为不选本身就是选择了浪费三个月。

#### 4. 选"即使错了也扛得住的"

| 选择 | 错了会怎样 | 扛得住吗 |
|------|-----------|---------|
| 全力备考，秋招只投央国企 | 省考没上岸，春招再找 | 扛得住，项目还在，培训班还在 |
| 全力找私企，不备考 | 找到工作但干得不开心 | 扛得住，可以边工作边考 |
| 继续纠结不选 | 什么都没做，秋招错过了 | 扛不住，两头空 |

#### 5. 培训班的话要辩证听

培训班说"坚持就一定能找到15万"——他们有他们的立场：就业率是命脉，我找到好工作=他们的广告，找不到他们最多说"再等等"，我浪费的是自己的时间。幸存者偏差：他们只看到坚持下来且成功的人。

#### 6. 项目做完≠必须去找工作

做完项目、拿到模拟面试机会，不代表必须走找工作这条路。项目是保底，不是枷锁。两万块买了个保底选项，不亏。

### 九、最终方案

**项目收尾 → 模拟面试 → 备考行测 → 央国企秋招 → 省考**

不摇摆，不改。

#### 详细时间线

| 时间 | 主线 | 说明 |
|------|------|------|
| 现在~6月中 | **做完项目**（3-5天） | 对得起两万块，也给自己保底 |
| 6月中 | **模拟面试** | 零成本验证真实水平，获得真实反馈 |
| 6月中~8月 | **全职备考行测**（每天6小时） | 资料分析→判断推理→言语理解→数量关系 |

| 9月~2027.3 | **全职备考行测+申论**（每天6小时） | 央国企面试/等结果 + 继续备考 |
| 11月 | **国考**（练手） | 能报就报，当实战演练 |
| 2027.3 | **省考**（核心目标） | 报有盒马的城市，限计算机专业的岗位 |
| 2027.3~6 | **如果都没上岸，春招找工作** | 项目还在，培训班还在，突击八股文2周 |

#### 秋招策略：只投央国企，不投私企

| | 央国企 | 私企 |
|---|---|---|
| 投不投 | **投** | 不投 |

| 对行测备考的影响 | 零影响 | 少1.5-2个月行测时间 |
| 保底 | 春招再投私企 | — |

不是"县区级公务员"，而是**有盒马的省会/二线城市**：

| 城市 | 盒马 | 体制内难度 | 生活成本 |
|------|------|-----------|---------|

| 沈阳 | 有 | 中等 | 中等 |
| 济南 | 有 | 中等 | 中等 |
| 长沙 | 有 | 中低 | 中等 |
| 合肥 | 有 | 中等 | 中等 |

### 十、备考具体规划

#### 行测学习顺序

1. **资料分析**（最先学，最容易提分，2-3周）
2. **判断推理**（3-4周）
3. **言语理解**（3-4周）
4. **数量关系**（2-3周，放在最后，性价比最低）
5. **常识**（碎片时间刷粉笔APP，不专门花大块时间）

#### 申论学习

- 每周至少完整写1篇大作文
- 先看课学方法，再动笔练
- 申论不是纯努力就能提分的，需要悟性和积累

#### 每日时间表

| 时间段 | 内容 | 目标 |
|--------|------|------|
| 上午 3h | 行测学习（看课+刷题） | 按模块推进 |
| 下午 3h | 行测刷题+错题总结 | 每天至少50题，错题记本上 |
| 晚上 2h | 申论（看课+动笔写） | 每周至少完整写1篇大作文 |
| 碎片时间 | 粉笔APP刷常识 | 每天通勤/吃饭时刷20题 |

#### 除了备考，还要做

1. **身体**：每天30分钟运动（跑步/快走/跳绳），运动是成本最低的抗焦虑方式
2. **信息收集**：每周1小时，关注黑龙江公务员考试网、银行招聘网，看往年岗位和分数线
3. **项目话术**：1周内整理好，五子棋和录屏项目各一段3分钟能讲完的介绍
4. **Java基础**：秋招前2周突击，不用现在学

#### 不要做的事

| 不要 | 原因 |
|------|------|
| 不要同时学技术和备考 | 精力分散，两边都半吊子 |
| 不要刷考公焦虑帖 | 网上全是贩卖焦虑，对我没帮助 |
| 不要跟别人比进度 | 有人备考2年没上岸，有人3个月上岸，没有可比性 |
| 不要纠结"万一考不上" | 最坏结果就是春招找工作，死不了 |
| 不要熬夜 | 每天8小时有效学习就够了，熬夜只会增加焦虑 |

### 十一、关于考公难度的诚实评估

#### 我之前说过的过于乐观的话

| 之前说的 | 修正后 |
|----------|--------|
| 实际竞争对手只有3-5人 | 5-15人（取决于当年岗位），限户籍的岗位不一定每年都有 |
| 成功率50-70% | 30-50%（认真备考的前提下），可能需要考1-2次 |
| 每天10小时坚持半年 | 大部分人做不到，按6-7小时规划更现实 |
| 行测努力就有分 | 大部分模块努力有用，但申论和常识看底子 |

#### 考公的真实风险

- 培训班很贵（线下1-3万），但我可以自学（网课500-1000元+书200-300元，一年1500块）
- 备考期间没有收入，但我还在学校，管吃住，经济压力几乎为零
- 如果省考没上岸，春招时技术更生疏了，但项目还在、培训班还在
- 面试环节有主观因素，不是笔试分高就一定上岸

#### 综合上岸率（体制内）

央国企秋招（40-50%）+ 省考（30-50%）+ 春招保底，**综合进入体制内的概率约60-70%。**

### 十二、给自己的提醒

1. **焦虑的解药是行动，不是想清楚。** 纠结三个月的问题，用行动结束它。
2. **考公失败的沉没成本远小于走错技术路的沉没成本。** 最坏结果不过是浪费几个月备考时间，春招还能找工作。
3. **有退路的人反而考得更好。** 对我这种容易焦虑的性格，保底方案能减少内耗，提高专注度。
4. **不要被"背水一战"绑架。** 那适合压力越大越兴奋的人，不适合我。
5. **行测申论比计算机进阶更适合我。** 高考514说明我基础在，行测是"努力就有分"的模式，反馈明确。
6. **两种困难，一种让你更迷茫，一种让你更清楚。** 面试失败你很难知道"我到底差在哪"，行测遇到瓶颈你知道"资料分析正确率60%，就练资料分析"。
7. **后悔是不可避免的，但"不选"才是最糟糕的选择。**
8. **两万块已经花了，回不来了。** 为了不浪费两万块而选一条不适合的路，代价远不止两万块。
9. **我想去有盒马的城市生活。** 这意味着我要考更高的分，目标定高一点，多考10-15分。
10. **模拟面试是零成本的试金石。** 做完项目就去，比纠结三个月有用一万倍。
