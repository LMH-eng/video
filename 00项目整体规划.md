# 短视频平台 - 项目整体规划

## 一、项目概述

基于FFmpeg+SDL+OpenCV+Qt的短视频平台，C/S架构，类似抖音。
- 客户端：Windows Qt项目（`e:\VideoPlayer\netdisk\NetDisk\`）
- 服务端：Linux epoll+线程池项目（虚拟机 `192.168.18.129`，代码路径 `/home/lmh/netdisk/0611/NetDisk/`）
- 流媒体服务器：Nginx-RTMP（虚拟机上已搭建）

---

## 二、项目架构

```
┌─────────────────── 客户端(Windows Qt) ───────────────────┐
│                                                           │
│  LoginDialog ──→ MainDialog                               │
│  (注册/登录)      ├── 播放器模块(VideoPlayer+SDL)          │
│                   ├── 录制模块(PicInPic_Read+Audio_Read)   │
│                   ├── 上传模块(UploadWorker+moveToThread)  │
│                   ├── 推荐列表(GIF缩略图)                  │
│                   └── 网络层(netapi/TCP客户端)              │
│                           ↓ TCP                           │
├─────────────────── 服务端(Linux) ─────────────────────────┤
│  TcpKernel → CLogic → CMysql                             │
│  (epoll+线程池)  (业务逻辑)  (数据库)                      │
│                           ↓                               │
├─────────────────── Nginx-RTMP ────────────────────────────┤
│  RTMP推流/拉流  VOD点播  HLS直播                           │
└───────────────────────────────────────────────────────────┘
```
