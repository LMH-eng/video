# 11 注册登录功能实现

## 一、功能概述

实现短视频平台的用户注册与登录功能，客户端（Windows Qt）与服务端（Linux epoll）通过TCP通信完成注册登录流程。

## 二、整体架构

```
客户端(LoginDialog) → CKernel → netapi(TCP) → 服务端(TcpKernel/CLogic) → MySQL
```

## 三、数据库设计

### MySQL建表语句

```sql
USE NetDisk;
CREATE TABLE IF NOT EXISTS t_UserData (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    tel VARCHAR(50) NOT NULL,
    password VARCHAR(50) NOT NULL,
    food INT DEFAULT 50,
    funny INT DEFAULT 50,
    music INT DEFAULT 50,
    game INT DEFAULT 50,
    sport INT DEFAULT 50,
    knowledge INT DEFAULT 50
) DEFAULT CHARSET=utf8;
```

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 用户唯一标识 |
| name | VARCHAR(50) | 昵称 |
| tel | VARCHAR(50) | 手机号（注册查重依据） |
| password | VARCHAR(50) | MD5加密后的密码 |
| food/funny/music/game/sport/knowledge | INT DEFAULT 50 | 爱好向量（后续推荐算法用） |

## 四、协议包定义（packdef.h）

### 注册请求
```cpp
#define _DEF_PACK_REGISTER_RQ 1
struct STRU_REGISTER_RQ {
    int type;           // = _DEF_PACK_REGISTER_RQ
    char tel[40];       // 手机号
    char name[40];      // 昵称
    char password[40];  // MD5加密后的密码
};
```

### 注册响应
```cpp
#define _DEF_PACK_REGISTER_RS 2
struct STRU_REGISTER_RS {
    int type;       // = _DEF_PACK_REGISTER_RS
    int result;     // register_success / user_is_exist
};
```

### 登录请求
```cpp
#define _DEF_PACK_LOGIN_RQ 3
struct STRU_LOGIN_RQ {
    int type;           // = _DEF_PACK_LOGIN_RQ
    char tel[40];       // 手机号
    char password[40];  // MD5加密后的密码
};
```

### 登录响应
```cpp
#define _DEF_PACK_LOGIN_RS 4
struct STRU_LOGIN_RS {
    int type;       // = _DEF_PACK_LOGIN_RS
    int result;     // login_success / user_not_exist / password_error
    int userid;     // 登录成功时返回用户id
};
```

## 五、客户端实现

### 5.1 LoginDialog（登录注册界面）

**文件**：`logindialog.h` / `logindialog.cpp` / `logindialog.ui`

**界面设计**：使用QStackedWidget实现两个页面切换

```
┌─── 登录页（page_login）──────────┐
│  用户登录                         │
│  手机号: [__________]             │
│  [红色错误提示]                    │
│  密码:   [__________] [显示]      │
│  [红色错误提示]                    │
│       [登录]                      │
│    没有账号？去注册                │
└──────────────────────────────────┘

┌─── 注册页（page_register）───────┐
│  用户注册                         │
│  手机号:   [__________]           │
│  [红色错误提示]                    │
│  昵称:     [__________]           │
│  [红色错误提示]                    │
│  密码:     [__________] [显示]    │
│  [红色错误提示]                    │
│  确认密码: [__________] [显示]    │
│  [红色错误提示]                    │
│       [注册]                      │
│    已有账号？去登录                │
└──────────────────────────────────┘
```

**校验规则（对标企业级）**：

| 校验项 | 规则 | 实现方式 |
|--------|------|----------|
| 手机号 | 11位纯数字，必须1开头 | QRegExpValidator + checkRegTel/checkLoginTel |
| 密码 | 4-20位 | checkRegPassword/checkLoginPassword |
| 确认密码 | 必须和密码一致 | checkRegConfirm |
| 昵称 | 2-20位 | checkRegName |
| 实时红字提示 | 输入时实时校验，红色文字显示在输入框下方 | QLabel + textChanged信号 |
| 按钮禁用 | 校验不通过时登录/注册按钮灰色不可点 | setEnabled(false/true) |
| 密码显示/隐藏 | 点击"显示/隐藏"按钮切换 | setEchoMode(Normal/Password) |
| 密码tooltip | 鼠标悬停显示"密码为4位到20位" | setToolTip |

**信号**：
- `SIG_RegisterRq(QString tel, QString name, QString password)` — 密码已MD5加密
- `SIG_LoginRq(QString tel, QString password)` — 密码已MD5加密
- `SIG_close()` — 关闭窗口

**MD5加密**：
```cpp
QByteArray pwdHash = QCryptographicHash::hash(password.toUtf8(), QCryptographicHash::Md5);
QString md5Pwd = QString(pwdHash.toHex());
```

### 5.2 CKernel（核心调度类）

**文件**：`ckernel.h` / `ckernel.cpp`

**新增成员**：
```cpp
LoginDialog* m_loginDialog;   // 登录注册界面

```

**启动流程**：
```
CKernel构造 → 创建netapi → 先显示LoginDialog → 登录成功后显示MainDialog
```

**关键代码**：
```cpp
// 构造函数中
m_loginDialog = new LoginDialog;
connect(m_loginDialog, SIGNAL(SIG_RegisterRq(...)), this, SLOT(slot_registerRq(...)));
connect(m_loginDialog, SIGNAL(SIG_LoginRq(...)), this, SLOT(slot_loginRq(...)));
m_loginDialog->show();
```

**注册槽函数**：
```cpp
void CKernel::slot_registerRq(QString tel, QString name, QString password)
{
    STRU_REGISTER_RQ rq;
    strcpy(rq.tel, tel.toUtf8().data());
    strcpy(rq.name, name.toUtf8().data());
    strcpy(rq.password, password.toUtf8().data());
    m_tcpClient->SendData(0, (char*)&rq, sizeof(rq));
}
```

**登录槽函数**：
```cpp
void CKernel::slot_loginRq(QString tel, QString password)
{
    STRU_LOGIN_RQ rq;
    strcpy(rq.tel, tel.toUtf8().data());
    strcpy(rq.password, password.toUtf8().data());
    m_tcpClient->SendData(0, (char*)&rq, sizeof(rq));
}
```

**处理服务端回复**：
```cpp
void CKernel::slot_dealClientData(unsigned int lSendIP, char *buf, int nlen)
{
    int type = *(int*)buf;
    switch(type)
    {
    case _DEF_PACK_REGISTER_RS:
        // 注册成功/用户已存在 → 弹窗提示
        break;
    case _DEF_PACK_LOGIN_RS:
        // 登录成功 → 隐藏LoginDialog，显示MainDialog，记录userid
        // 用户不存在/密码错误 → 弹窗提示
        break;
    }
}
```

## 六、服务端实现

### 6.1 CLogic::RegisterRq

```cpp
void CLogic::RegisterRq(sock_fd clientfd, char* szbuf, int nlen)
{
    STRU_REGISTER_RQ* rq = (STRU_REGISTER_RQ*)szbuf;
    STRU_REGISTER_RS rs;

    // 1. 查重：手机号是否已存在
    char sqlBuf[_DEF_SQLIEN] = "";
    sprintf(sqlBuf, "SELECT id FROM t_UserData WHERE tel='%s';", rq->tel);
    list<string> lstRes;
    if(m_sql->SelectMysql(sqlBuf, 1, lstRes))
    {
        if(lstRes.size() > 0)
        {
            rs.result = user_is_exist;
            SendData(clientfd, (char*)&rs, sizeof(rs));
            return;
        }
    }

    // 2. 不存在，插入数据库
    sprintf(sqlBuf, "INSERT INTO t_UserData(name, tel, password) VALUES('%s','%s','%s');",
            rq->name, rq->tel, rq->password);
    rs.result = m_sql->UpdataMysql(sqlBuf) ? register_success : user_is_exist;

    SendData(clientfd, (char*)&rs, sizeof(rs));
}
```

### 6.2 CLogic::LoginRq

```cpp
void CLogic::LoginRq(sock_fd clientfd, char* szbuf, int nlen)
{
    STRU_LOGIN_RQ* rq = (STRU_LOGIN_RQ*)szbuf;
    STRU_LOGIN_RS rs;

    // 1. 查手机号
    char sqlBuf[_DEF_SQLIEN] = "";
    sprintf(sqlBuf, "SELECT id, password FROM t_UserData WHERE tel='%s';", rq->tel);
    list<string> lstRes;
    if(m_sql->SelectMysql(sqlBuf, 2, lstRes))
    {
        if(lstRes.size() == 0)
        {
            rs.result = user_not_exist;
            SendData(clientfd, (char*)&rs, sizeof(rs));
            return;
        }

        string strId = lstRes.front(); lstRes.pop_front();
        string strPwd = lstRes.front(); lstRes.pop_front();

        // 2. 验证密码
        if(strcmp(strPwd.c_str(), rq->password) != 0)
        {
            rs.result = password_error;
            SendData(clientfd, (char*)&rs, sizeof(rs));
            return;
        }

        // 3. 登录成功
        rs.result = login_success;
        rs.userid = atoi(strId.c_str());
    }
    else
    {
        rs.result = user_not_exist;
    }

    SendData(clientfd, (char*)&rs, sizeof(rs));
}
```

## 十、踩坑记录

1. **config.ini旧IP**：程序启动时会读取exe同目录下的config.ini，如果里面存着旧IP会导致连接超时、界面出来很慢。解决：修改config.ini中的IP为虚拟机实际IP。
2. **OpenNet参数类型**：`INetMediator::OpenNet` 接受 `const char*`，不能直接传QString，需要 `m_ip.toUtf8().data()` 转换。
3. **QCloseEvent不完整类型**：需要在logindialog.cpp中 `#include <QCloseEvent>`。
4. **QRegExpValidator头文件**：需要 `#include <QRegExpValidator>`，单独包含QRegExp不够。
